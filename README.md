
# FESOM2 Accelerated

Simple C routines to offer native GPU acceleration to [FESOM2](https://github.com/FESOM/fesom2/).

# Dependencies

- [CUDA](https://developer.nvidia.com/cuda-downloads)
- [kernel_tuner](https://github.com/benvanwerkhoven/kernel_tuner)
